# Electrical Aptitude Test Prep

A simple static website for US-based electrical apprenticeship aptitude test practice. This site provides free practice exams and study resources to help prepare for electrical apprenticeship aptitude tests.

## What is this?

This is a static website containing practice exams, math problems, and reading comprehension questions designed to help people prepare for electrical apprenticeship aptitude tests (such as the IBEW test).

## Running Locally

Since this is a static website, you can easily download and run it on your computer without any special software or setup.

### Option 1: Simple Method (Recommended)

1. **Download the files**: Click the green "Code" button on this page, then click "Download ZIP"
2. **Extract the ZIP file**: Unzip the downloaded file to a folder on your computer
3. **Open the website**: 
   - Navigate to the extracted folder
   - Double-click on `index.html` to open it in your web browser
   - That's it! The website should now be running in your browser



## Notes

- All files are static HTML, CSS, and JavaScript - no database or server required
- The website works entirely in your browser once opened
- You can bookmark `index.html` for easy access
- All practice questions and answers are included in the HTML files
